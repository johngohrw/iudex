# Skill: write-prd

**Purpose:** Produce a well-structured PRD from a grilled idea or requirement.

## How to use
Tell Claude: "Use write-prd to write a PRD for: [idea or grill-me output]"

## Output format
Save as `docs/prd/<feature-name>.md`:

```markdown
# PRD: <Feature Name>

## Summary
One paragraph. What this is and why it exists.

## Background & Motivation

## Goals
- ...

## Non-Goals
- ...

## User Stories
- As a <user>, I want <action> so that <outcome>.

## Functional Requirements
1. ...

## Technical Constraints
- ...

## Open Questions
- ...

## Success Metrics
- ...
```
